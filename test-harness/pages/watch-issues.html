Watch Mode Issues Fixture

Emits predictable console errors and failing network requests on page load. Used by harness block [78] to validate WatchSession poll, dedup, and incremental detection.

On Load (first poll captures these)

  • console.error — ARGUS_WATCH_TEST: console error alpha
  • console.warn — ARGUS_WATCH_TEST: console warning beta
  • Fetch /api/always-500 → HTTP 500 (network_server_error)
  • Fetch /api/missing → HTTP 404 (network_not_found)

Incremental triggers (called from test code)

  • window.argusWatchTriggerError(msg) — fires a new console.error
  • window.argusWatchTrigger404() — fires a new 404 request
